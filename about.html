About the Project

This small project was created as a front-end demo for the March 2021 cohort of Digital Crafts Full Stack Bootcamp. It was created using Vanilla JS, HTML/CSS, and a lovely CSS library called skeleton css. For more information, please check out the readme on GitHub s and the Medium article.

Creator

Patrick Groves is a software developer at Digtal Crafts Bootcamp. His previous life was in the hospitality industry, and he still cooks excellent scratch meals for his wife. When not coding or dreaming of applications, Patrick also likes to hike and compose killer riffs on his bass and cello that no one will ever hear.
